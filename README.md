# LinuC102 学習用サイト

LinuC102試験対策用のインタラクティブな学習ウェブサイトです。

## 機能

- **模擬試験**: リアルな試験問題を時間制限で解答
- **問題集**: カテゴリ別に分類された問題
- **進捗管理**: 学習進度の追跡と統計
- **復習機能**: 間違えた問題を重点的に復習

## ディレクトリ構成

```
├── index.html          # メインページ
├── styles/
│   └── style.css       # スタイルシート
├── js/
│   ├── app.js          # メインアプリケーション
│   ├── questions.js    # 問題データベース
│   └── utils.js        # ユーティリティ関数
├── data/
│   └── questions.json  # 問題データ
└── README.md           # このファイル
```

## 使い方

1. `index.html` をブラウザで開く
2. 学習モードまたは模擬試験を選択
3. 問題に答える
4. 進捗を確認

## 対応範囲

- LinuC102試験の全範囲
- 101試験からのステップアップ項目
- 実務的な問題解決スキル
